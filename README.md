# ML Course Homeworks

## Требования
- Python 3.10+
- [uv](https://docs.astral.sh/uv/getting-started/installation/)

## Установка

1. Клонируй репозиторий:
```bash
   git clone 
   cd ml-course-homeworks
```

2. Установи зависимости:
```bash
   uv sync
```

3. Активируй виртуальное окружение:
```bash
   # Windows
   .venv\Scripts\activate

   # Linux/macOS
   source .venv/bin/activate
```
